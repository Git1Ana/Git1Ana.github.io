Анализатор востребованных навыков

О проекте

Веб-приложение для автоматизации анализа востребованных навыков на рынке труда. Приложение позволяет значительно сократить время, необходимое соискателям и учебным заведениям для оценки текущих требований работодателей и определения ключевых навыков.

Ключевые результаты:

Сокращение времени сбора вакансий: > 90%
Сокращение времени анализа компетенций: > 70%

Основные функции

  • Интеграция с API HeadHunter для сбора актуальных вакансий в реальном времени
  • Гибкая система фильтрации вакансий по различным критериям
  • Интеграция с Yandex GPT для автоматической классификации и анализа навыков
  • Визуализация данных в виде интерактивных таблиц и графиков связей
  • Анализ зависимости навыков от уровня заработной платы
  • Сохранение результатов анализа и возможность их объединения
  • Экспорт отчетов в текстовом формате
  • Управление кастомными промптами для ИИ-анализа

Использованные технологии

Python Flask PostgreSQL JavaScript HTML/CSS HeadHunter API Yandex GPT API Vis.js REST API

Что показано в демонстрации

В видео показан полный процесс работы с приложением: от настройки параметров поиска вакансий и сбора данных до анализа навыков с помощью ИИ, визуализации результатов и экспорта отчетов.

  1. Настройка фильтров поиска вакансий
  2. Сбор и обработка данных с HeadHunter
  3. Анализ навыков с использованием Yandex GPT
  4. Визуализация связей между навыками
  5. Сравнение различных выборок данных
  6. Экспорт результатов анализа
